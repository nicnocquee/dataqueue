Welcome to the DataQueue docs! Start from the [Introduction](/intro) to learn more about DataQueue.

DataQueue supports **PostgreSQL** and **Redis** as storage backends. Choose the one that fits your stack -- the API is identical regardless of which backend you use.

## Source Code

The source code for DataQueue is available on [GitHub](https://github.com/nicnocquee/dataqueue).

## Demo App

A demo Next.js app that showcases all features of DataQueue is available on [GitHub](https://github.com/nicnocquee/dataqueue/tree/main/apps/demo).
